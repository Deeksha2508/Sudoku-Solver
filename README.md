# Sudoku-Solver
This project is a Python-based Sudoku Solver that automatically solves a given Sudoku puzzle using the Backtracking Algorithm. Sudoku is a logic-based number puzzle consisting of a 9×9 grid divided into smaller 3×3 subgrids. The objective is to fill the grid so that each row, column, and 3×3 subgrid contains all digits from 1 to 9 exactly once.
